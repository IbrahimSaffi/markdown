# React Markdown Editor

![](./screenshots/markdown.jpg)


Markdown editor should be able to parse
- Heading 1 (#)
- Heading 2 (##)
- Heading 3 (###)
- List item (-)
- Horizontal Rule (`---`)
- Bold text (`**text**`)
- BlockQuote (`>`)
- Code Block (```)
